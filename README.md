# PanelForge

PanelForge is a visual comic panel layout workspace for building manga pages, storyboard sheets, Webtoon drafts, and illustration collages.

It helps creators quickly generate panel grids, adjust dynamic slanted cuts, place images inside panels, add speech bubbles and sound-effect stickers, tune per-panel visual effects, and export the finished layout as SVG or high-resolution PNG.

## What It Does

- Generate comic panels in portrait, square, landscape, or vertical Webtoon formats.
- Randomize panel layouts or apply preset templates for fast composition.
- Adjust panel count, gutter width, border width, background color, and slant intensity.
- Select a single panel and customize its image, background, transparency, and effect settings.
- Upload local images and drag them into comic panels.
- Add speech bubbles, sound-effect text, and custom sticker images.
- Drag visual elements directly on the canvas for precise placement.
- Export the result as editable SVG or high-resolution PNG.

## Who It Is For

PanelForge is designed for comic artists, illustrators, storyboard creators, designers, and students who need a fast way to explore page composition before moving into final drawing or publishing.

It is especially useful for:

- Manga and comic page drafts
- Webtoon layout planning
- Visual storyboarding
- Social media comic panels
- Illustration collage layouts
- Exportable composition references

## Basic Workflow

1. Choose the canvas ratio.
2. Generate a random layout or apply a preset template.
3. Tune panel count, slant, gutters, borders, and colors.
4. Select a panel and add images, bubbles, stickers, and effects.
5. Drag elements on the canvas to refine the composition.
6. Export the final work as SVG or PNG.
